Vanilla Roles

These roles were ported as closely as the phyiscal version of the game as I could/was willing to do. Since many Roles gave the player a one-time use ability, most of them were tweaked to either give a Status Effect or a Card at the beginning of the game for ease of implementation, but the end result should be roughly the same. Cards that are given as a result of being assigned a role are not placed in the Discard Pile after being used, to avoid disrupting the balance of the Deck. Roles that have been changed from the tabletop version are marked with an asterisk (*)

Human Roles

Roles that can be assigned to Players on the Human team.

Captain

You start with 1 stack of the Sedated Status Effect.

Pilot

You start with 1 Cat card.

Copilot

You start with 1 Teleport card.

Soldier

You start with 1 Attack card.

Psychologist

You start the game in a randomly selected Alien Start Sector.

Executive Officer

You start with 1 stack of the Lurking Status Effect

Medic

You start with 1 Scanner card.

Engineer

You permanently gain the Knowhow Status Effect.

Alien Roles

Roles given to Players on the Alien Team. In the tabletop version, many of the Alien roles feel (to me) quite underwhelming, so for any Alien whose Role allowed them to use a certain card, I've adjusted them to start with 1 of those cards, but they cannot draw any more. Hopefully, this will bring them in line with the Human Roles

Fast Alien

You start with 1 stack of the Adrenaline Surge Status Effect.

Surge Alien

You start with 1 Adrenaline card.

Blink Alien

You start with 1 Teleport card. I am aware of how broken this currently is, it's a side effect of the way I've adjusted Alien Roles and I'm working on balancing it.

Silent Alien

You start with 1 Sedatives card.

Brute Alien

You permanently gain the Armored Status Effect.

Invisible Alien

You permanently gain the Invisible Status Effect.

Lurking Alien

You permanently gain the Lurking Status Effect.

Psychic Alien

You permanently gain the Deceptive Status Effect.
