The Map Editor

EFAiOS Online comes with a built-in map editor for you to make any map you can think of. This page explains how to use it.

Overview

To access the map editor for creating new maps, you can click here, or the link to the Map Editor can always be found at the top of the screen in the navigation bar. Alternatively, to edit an existing map, you can click the "Edit" button next to any map on the Maps page.

When you load the Map Editor, you'll see a screen like this:

On the left is the sidebar, where you can configure the map details and settings, as well as your toolbar for selecting various sector types you can place in the map. The right side is your map preview.

Map Details

These fields are what others will see when your map is saved and appears on the Maps list:

  • Name: The name of the map as it appears at the top of the list entry on the Maps page, such as Galelei, Fermi, or Galvani.
  • Description: The description that appears between the map name and the Select & Edit buttons. This field accepts raw HTML, so you can structure the description however you want, including using bold, italics, and various text sizes.

Map Layout

This is where most of the magic happens!

Tools

This first section is your toolbar, which includes 6 different tools for you to choose from, as well as an indicator of which tool is currently selected. When any tool is selected, you can click any sector on the Map Preview (to the right) to set that sector to the type selected by the tool.

Rows & Columns

Using these two controls, you can adjust how many rows and columns your map has. Note that adjusting these values does not affect the map on its own - you must use the "Clear and Rebuild" button for the rows and columns to update. This also means that any work you have will be lost.

Map Config

Pressing this button will open a Game Config popup, just like what happens when you click the "Game Settings" button in a pre-game lobby. However, this config editor works slightly differently; The config set in this popup will be the default configuration any time someone starts a Lobby using this map. They can adjust the settings to their liking, of course, but using this, you can suggest a certain configuration for those playing your map.

Save

When you're happy with your map (or just want to save it and come back to editing later, after playtesting it, for example) you can hit Save to save it to whatever filesystem is running the server. When a map is saved, it is saved as a .json file in the /maps folder wherever you're running the server from. This can be useful if you want to send your map to someone else!

Note: In order for a Map to be valid to save, it must have at least 1 Human Start and at least 1 Alien Start. It can have multiple of any of these, but it must have at least one.

Tips & Tricks

  • Your map can have as many of any Sector type as you like, but be aware: When having multiple Human Starts or Alien Starts, at the beginning of the game, players on that team will be randomly placed in a sector belonging to their team - there is currently no way to control which players start in which of their team's Start Sectors.
  • The closer together you have Human and Alien starts, the more ambiguous you should allow early Human moves to be. This avoids Aliens being able to consistently and easily kill Humans within the first few turns.
  • While the map editor supports any size of map, remember that the bigger a map is, the longer games will generally take when playing that map.
  • You should absolutely play around with having varying numbers of Start Sectors and Escape Pods. Variety is the spice of life, after all!
  • If you're feeling stumped on what to make, try looking at some of the official maps for ideas - ask yourself what you like about them, or what you'd change.
